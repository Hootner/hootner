# 🎮 Quick Start Guide - New Features

## ⚡ Command Palette (Ctrl + K)

**What it does:** Professional command launcher for instant access to any feature

**How to use:**
1. Press `Ctrl + K` anywhere in the Dashboard
2. Type to search (e.g., "theme", "analytics", "notifications")
3. Click or press Enter to execute command
4. Press ESC to close

**Available Commands:**
```
🎨 Toggle Theme          - Switch between dark/light mode
🔔 Open Notifications    - View recent alerts
⌨️ Show Shortcuts        - Display keyboard shortcuts
⚡ Quick Actions         - Open quick actions panel
🚀 Platform Navigation   - Access all 22 platform pages
📝 Create New Post       - Create content (coming soon)
📊 View Analytics        - Jump to analytics tab
🛡️ Open Moderation      - Jump to moderation tab
⚙️ System Settings       - Jump to system tab
```

**Pro Tips:**
- Search is fuzzy: "notif" finds "Open Notifications"
- No mouse needed: keyboard-only workflow
- Recent commands appear first
- Commands auto-close after execution

## ⚡ Quick Actions (Ctrl + Q)

**What it does:** One-click operations for common tasks

**How to use:**
1. Press `Ctrl + Q` anywhere in the Dashboard
2. Click any action card
3. See instant feedback via toast notification

**6 Quick Actions:**
```
📥 Export Data        - Download platform data
💾 Create Backup      - Create system backup
🗑️ Clear Cache        - Clear browser cache
🔄 Refresh All        - Refresh all data
📄 Generate Report    - Create analytics report
⚙️ Quick Settings     - Access quick settings
```

**Visual Feedback:**
- Hover animation (cards lift and glow)
- Toast notification confirms action
- Color-coded icons for quick identification

## 📊 Performance Monitor

**What it does:** Real-time system health monitoring

**Location:** Bottom-right corner of Dashboard (always visible)

**3 Metrics Displayed:**
```
FPS: 60        ← Frame rate (green ≥55, yellow <55)
RAM: 45MB      ← Memory usage (Chrome only)
Ping: 23ms     ← Simulated latency
```

**Why it matters:**
- **FPS < 55**: Performance degradation, reduce tabs/animations
- **RAM > 200MB**: High memory usage, consider refresh
- **Ping > 100ms**: Network issues, check connection

**Auto-updates:** Every second, no performance impact

## 🐛 Troubleshooting

### **Command Palette Not Opening**
- Ensure Ctrl + K is not blocked by browser
- Try refreshing page (Ctrl + Shift + R)
- Check if modal is already open (press ESC first)

### **Quick Actions Not Responding**
- Check if toast notification appeared
- Some actions show "Coming soon" message
- Verify network connection for backend actions

### **Performance Monitor Shows 0**
- RAM metric only works in Chrome
- FPS needs time to stabilize (wait 2-3 seconds)
- Ping is simulated (not real network ping)

### **Animations Not Smooth**
- Check FPS metric (should be 55-60)
- Disable browser extensions
- Update graphics drivers
- Enable hardware acceleration in browser
