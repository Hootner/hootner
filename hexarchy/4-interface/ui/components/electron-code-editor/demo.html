📋 Usage Examples

// Initialize
import './ai-agent-init.js';

// Send message
window.aiAgentUI.sendMessage('Refactor this code');

// Toggle panel
window.aiAgentUI.toggle();

// Show analytics
window.aiAgentUI.showAnalytics();

✨ Features

🌐 Multi-Agent

6 specialized agents: All, Refactor, Debug, Optimize, Security, Test

💬 Streaming

Real-time token-by-token response display

🎤 Voice Input

Speech-to-text integration for hands-free operation

📎 Attachments

Add context files to your conversations

📊 Analytics

Track messages, tokens, sessions, and plugins

🔀 Diff Viewer

Side-by-side code comparison tool

🎨 Themes

4 themes: Dark, Light, Cyberpunk, Forest

💾 Export

Save and restore complete sessions

⌨️ Keyboard Shortcuts

Ctrl+Shift+A → Toggle panel
Ctrl+K → Focus input
Enter → Send message
Shift+Enter → New line
↑ → Recall last message

🔌 Plugin System

aiAgentUI.registerPlugin('myPlugin', {
name: 'My Plugin',
action: (input) => input.toUpperCase()
});

👥 Collaborative Mode

Multiple agents work together on your request
