# 🚀 GitBook Quick Start

## 3-Minute Setup

### Step 1: Create Account (1 min)
```
1. Go to: https://www.gitbook.com
2. Click "Sign up with GitHub"
3. Authorize GitBook
```

### Step 2: Import Docs (1 min)
```
1. Click "Create a new space"
2. Choose "Import from GitHub"
3. Select your HOOTNER repository
4. Click "Import"
```

### Step 3: Publish (1 min)
```
1. Review your docs
2. Click "Publish" (top right)
3. Copy your docs URL
4. Share with team!
```
