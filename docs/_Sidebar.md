# Sidebar Component

## Overview

Collapsible navigation sidebar for HOOTNER with quantum-grade styling.

## Location

`apps/frontend/src/components/Sidebar.tsx`

## Features

- Collapsible toggle (responsive: 64px → 256px/288px/320px)
- Fixed position navigation
- Quantum gradient background
- Icon-based navigation
- Hover effects

## Navigation Items

- 🏠 Dashboard
- 📹 Videos
- 📊 Analytics
- 🤖 AI Studio
- ⚙️ Settings

## Usage

```tsx
import Sidebar from './components/Sidebar'

<Sidebar />
```

## Behavior

- Click arrow to collapse/expand
- Icons always visible
- Labels show when expanded
- Fixed to left edge
