# 🦉 HOOTNER Commands Reference

This directory contains all commands organized by category for easy reference and execution.

## 📁 Structure

```
commands/
├── npm-scripts.md          # All npm/package.json scripts
├── docker-commands.md      # Docker & Docker Compose commands
├── development.md          # Development workflow commands
├── testing.md             # Testing & QA commands
├── deployment.md          # Deployment & CI/CD commands
├── security.md            # Security & audit commands
├── maintenance.md         # Backup, monitoring & maintenance
├── ai-services.md         # AI & video generation commands
└── quick-reference.md     # Most commonly used commands
```

## 🚀 Quick Start Commands

```bash
# Start development
npm run dev

# Start video player
cd apps/frontend/html-pages && node video-player-server.js

# Run all tests
npm test

# Security audit
npm run security:audit

# Docker development
npm run docker:up
```

## 📖 Usage

Each file contains:
- ✅ **Command** - The actual command to run
- 📝 **Description** - What the command does
- 📍 **Location** - Where to run it from
- ⚙️ **Prerequisites** - What's needed before running
- 🔗 **Related** - Related commands or workflows

Navigate to specific files for detailed command references.
