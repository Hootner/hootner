### Recording Setup
- **Screen Recording:** OBS Studio (1920x1080, 60fps)
- **Audio:** Blue Yeti / Rode NT-USB
- **Editor:** DaVinci Resolve / Premiere Pro
- **Code Font:** Fira Code, size 16
- **Theme:** Dark mode (easier on eyes)

### Video Structure (Template)
```

```
