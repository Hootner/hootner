## ⚠️ Medium-Priority Recommendations

### 1. Environment Variable Security

**Current:** JWT_SECRET has fallback value in code
**Risk:** Medium - Exposes secret in source control
**Recommendation:**

```javascript
// Current (subscriptions.js line 26)
jwt.verify(token, process.env.JWT_SECRET || 'fallback-secret');

// Recommended
const JWT_SECRET = process.env.JWT_SECRET;
if (!JWT_SECRET) {
  throw new Error('JWT_SECRET environment variable is required');
}
jwt.verify(token, JWT_SECRET);
```

### 2. CSP 'unsafe-inline' Restriction

**Current:** scriptSrc and styleSrc allow 'unsafe-inline'
**Risk:** Medium - Reduces XSS protection effectiveness
**Recommendation:** Implement nonce-based CSP or move to external files

### 3. Password Validation Enhancement

**Current:** Basic requirements defined but not enforced everywhere
**Recommendation:** Add password strength meter and enforce across all forms
