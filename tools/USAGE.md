# Tools Usage Examples

## Deploy Service
```bash
node tools/deploy-service.js prod frontend
```

## Chaos Test
```bash
node tools/chaos-test.js load-test 10
```

## System Health
```bash
node tools/system-health.js all
```

## Backup System
```bash
node tools/backup-system.js full
```

## Security Scan
```bash
node tools/security-scan.js dependencies
```
