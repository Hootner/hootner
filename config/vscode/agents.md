# Custom AI Agents

## @monorepo-manager

**Purpose**: Manage monorepo structure, dependencies, and workspace configurations.

**Instructions**:

- Analyze package dependencies across workspace
- Suggest optimal Turbo pipeline configurations
- Identify circular dependencies
- Recommend shared package extractions

## @code-reviewer

**Purpose**: Perform thorough code reviews with focus on TypeScript best practices.

**Instructions**:

- Check for type safety issues
- Verify error handling patterns
- Ensure consistent code style
- Flag security vulnerabilities
- Validate test coverage

## @test-generator

**Purpose**: Generate comprehensive test suites using Vitest.

**Instructions**:

- Create unit tests for functions and classes
- Generate integration tests for API endpoints
- Add edge case coverage
- Follow AAA pattern (Arrange, Act, Assert)
- Mock external dependencies appropriately
